About Engine Script

Engine Script is a scripting language written specifically for this engine. It allows creators to write clean, readable code.

On the object you want to add a script to, right-click the script cell and choose "New Script"
Step 1.
Take note of the path to the object you want to add a script to.
this path should look like DUNGEON_ID/path_to_object.json

Step 2.
Create a new file at DUNGEON_ID/scripts/path_to_object/
The file can be named anything, but it is reccommended to name it after the event name.
Only use characters a-z 0-9 and underscore for file names

Step 3.
Back in your object's json file, put the path to your new script file in the event you want.

Example:
"on_enter": {
"#script": "PATH_TO_SCRIPT" // It does not matter if you include the file extension or not.
}
